# 下学期课程内容更新计划

## 1. 核心目标
将研究小组在**大规模图处理**、**云端存储优化**、**分布式系统管理**以及**生成式 AI (RAG/LLM)** 领域的最新研究成果，转化为教学案例，提升《数据中心技术》与《计算机系统设计》课程的前沿性与深度。

## 2. 《数据中心技术》课程更新方案

### 现有内容回顾
- 数据中心概论、对象存储 (Object Storage)、Ceph 部署、QoS 基础。

### 新增/强化模块
| 模块名称 | 结合研究工作 | 教学重点 |
| :--- | :--- | :--- |
| **云端存储性能管理** | CoFS (2024), Parallelism vs Fairness (2024) | 探讨在多租户云环境下，如何平衡 SSD 的吞吐量(Parallelism)与用户公平性(Fairness)。 |
| **分布式存储自动化运维** | Rethinking Parameter Tuning (2026) | 引入“知识图谱驱动系统配置”的概念，展示如何从传统的手动调优转向智能化的自动化管理。 |
| **异构存储与 QoS** | Heterogeneous Object Storage (2026) | 讨论在包含多种介质的异构存储系统中，如何利用时空图模型实现精细化的 QoS 保障。 |
| **Serverless 与边缘计算** | Cold Start Optimization (2025) | 分析 Serverless 架构下的资源调度瓶颈，特别是冷启动问题的预测与缓解策略。 |
| **AI 基础设施演进** | CoT-RAG (2025), DGX Spark (新闻) | 讨论 RAG 技术如何改变数据中心的数据流向，以及本地 AI 超算对数据中心架构的影响。 |

## 3. 《计算机系统设计》讲座更新方案

### 现有内容回顾
- C 语言基础、图系统基础。

### 新增/强化模块
| 模块名称 | 结合研究工作 | 教学重点 |
| :--- | :--- | :--- |
| **高性能图处理设计** | SOWalker (2023), DistGER-Pipe (2025) | 深入探讨大规模图数据的 Out-of-core 处理设计、I/O 优化及分布式流水线设计。 |
| **现代存储控制器设计** | SSD Address Allocation (2024), Cache Replacement (2022) | 讲解 SSD 内部地址映射机制的优化，以及如何利用强化学习设计智能缓存替换算法。 |
| **系统级智能化** | Knowledge Graph Query (2026) | 展示如何将高层语义（知识图谱）引入底层系统设计，实现更智能的参数决策。 |
