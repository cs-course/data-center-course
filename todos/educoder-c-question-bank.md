### 【知识点：控制结构】！

**题19** `[基础]` `if语句`  
执行以下代码：  
```c
int x = 5;
```  

屏幕输出是（ ）  
A. `A`  
B. `AB`

**正确答案：B**  
> `if` 只控制第一条语句，`printf("B")` 始终执行。

**题20** `[基础]` `switch语句`  
执行以下代码：  
```c
int x = 2;
switch (x) {
    case 1: printf("A");
    case 2: printf("B");
    case 3: printf("C");
    default: printf("D");
}
```  
屏幕输出是（ ）  
A. `B`  
B. `BCD`  
C. `A`  
D. 编译错误  
**正确答案：B**  
> `switch` 没有 `break` 会继续执行后续 `case`（穿透）。

**题21** `[中等]` `for循环`  
执行以下代码：  
```c
int s = 0, i;
for (i = 1; i <= 5; i += 2) {
    s += i;
}
printf("%d", s);
```  
屏幕输出是（ ）  
A. `9`  
B. `6`  
C. `12`  
D. `15`  
**正确答案：A**  
> `i` 取值：1、3、5，`s = 1 + 3 + 5 = 9`。

**题22** `[中等]` `while循环`  
执行以下代码：  
```c
int x = 10;
while (x > 0) {
    if (x % 3 == 0) break;
    x--;
}
printf("%d", x);
```  
屏幕输出是（ ）  
A. 0  
B. 3  
C. 9  
D. 10  
**正确答案：B**  
> `x` 从10递减，当 `x = 9` 时 `9 % 3 == 0` 成立，`break` 跳出循环，输出 `9`。

**题23** `[较难]` `嵌套循环`  
执行以下代码：  
```c
int i, j, count = 0;
for (i = 1; i <= 3; i++) {
    for (j = 1; j <= i; j++) {
        count++;
    }
}
printf("%d", count);
```  
屏幕输出是（ ）  
A. 3  
B. 6  
C. 9  
D. 10  
**正确答案：B**  
> 内层循环次数：`i=1`时1次，`i=2`时2次，`i=3`时3次，共 `1+2+3=6` 次。

**题R6** `[较难]` `递归函数`  

```c
#include <stdio.h>
int f(int n) {
    if (n == 0) return 0;
    return f(n - 1) + n * n;
}
int main() {
    printf("%d\n", f(3));
    return 0;
}
```  

**输出结果：**  
```
14
```  
> 解释：`f(3) = f(2) + 9 = f(1) + 4 + 9 = f(0) + 1 + 4 + 9 = 0 + 1 + 4 + 9 = 14`

| 大题 | 题型 | 库中题量 | 每套抽题 | 分值/题 | 小计 |
|:---|:---|:---:|:---:|:---:|:---:|
| 一 | 单选题 | 40 | 15 | 2分 | 30分 |
| 二 | 程序阅读题 | 12 | 4 | 5分 | 20分 |
| 三 | 程序填空题 | 10 | 3 | 5分 | 15分 |
| 四 | 编程题 | 6 | 2 | 15分 | 30分 |
| **合计** | | **68** | **24** | | **95分** |

> 注：剩余5分为平时成绩/实验成绩，由任课教师另行评定。

### 难度分布要求（每套试卷）

| 难度 | 单选题(15) | 阅读题(4) | 填空题(3) | 编程题(2) |
|:---|:---:|:---:|:---:|:---:|
| 基础 | 7~8 | 1 | 1 | 0 |
| 中等 | 5~6 | 2 | 1 | 1 |
| 较难 | 2~3 | 1 | 1 | 1 |

### 组卷脚本示例（Python）

```python
import random

# 题目按难度分组
easy_choice   = [1,2,3,4,5,7,9,10,13,15,16,19,20,24,27,28,31,34,37]  # 19题
mid_choice    = [6,8,11,12,14,17,18,21,22,25,26,29,30,32,33,35,36,38]  # 18题
hard_choice   = [23,39,40]                                                # 3题

easy_read     = ['R1','R2']           # 2题
mid_read      = ['R3','R4','R5','R8','R11']  # 5题
hard_read     = ['R6','R7','R9','R10','R12']  # 5题

easy_fill     = ['F1','F2','F3']      # 3题
mid_fill      = ['F4','F5','F6','F7'] # 4题
hard_fill     = ['F8','F9','F10']     # 3题

mid_prog      = ['P1','P2','P3']      # 3题
hard_prog     = ['P4','P5','P6']      # 3题

def generate_paper():
    choice = (random.sample(easy_choice, 8) +
              random.sample(mid_choice, 6) +
              random.sample(hard_choice, 1))
    read = (random.sample(easy_read, 1) +
            random.sample(mid_read, 2) +
            random.sample(hard_read, 1))
    fill = (random.sample(easy_fill, 1) +
            random.sample(mid_fill, 1) +
            random.sample(hard_fill, 1))
    prog = (random.sample(mid_prog, 1) +
            random.sample(hard_prog, 1))
    return {'choice': sorted(choice), 'read': read,
            'fill': fill, 'prog': prog}

# 生成3套等价试卷
papers = [generate_paper() for _ in range(3)]
for i, p in enumerate(papers, 1):
    print(f"试卷{i}: {p}")
```
